## Forkify QA Testing Project

## 📌 Project Overview
This repository contains manual QA testing results for the Forkify web application. The bugs were identified and documented using Jira during exploratory and functional testing.

## 🧪 Testing Type
- Manual Testing
- Exploratory Testing
- Functional Testing

## 🐞 Bug Report Summary
- Total Bugs Reported: 18
- All bugs were logged and managed in Jira
- Exported bug report is included as a CSV file in this repository

## 🛠 Tools Used
- Jira (for bug tracking and reporting)
- Browser-based manual testing

## 📁 Files Included
- Jira CSV Export File
  → Contains all bug reports with details such as summary, description, severity, and steps to reproduce

## 🎯 Objective
The goal of this project was to practice real-world QA testing by identifying, logging, and managing defects in Jira and exporting structured reports for documentation.

## 📌 Notes
This project is part of QA practice work to improve skills in bug reporting, Jira usage, and manual testing techniques.
